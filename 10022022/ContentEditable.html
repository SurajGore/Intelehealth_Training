The browser have a nifty new attribute that can be applied to elements, called contenteditable. As the name impiles, this allows the user to edit any of the text contained within the element, including its children. There are a variety of uses for something like this, including an app as simple as a to-do list, which also of local stroage. The NOte: When The contenteditable attribute is not set on an element, the element will inherit it from its parent.
